Beta build 13 improves sign-in and session handling so the app behaves more predictably when restoring saved accounts or refreshing an expired login.

More Reliable Sessions

  • Saved sessions are restored more reliably when you reopen the app.
  • Areas that depend on an active account, including project loading, billing, and chat actions, now wait for a valid session before continuing.
  • Signing out clears local session state more consistently, which reduces stale account data lingering in the interface.

Notes

  • If you had an older cached session, you may be asked to sign in again once after updating.
  • No project migration is required.
