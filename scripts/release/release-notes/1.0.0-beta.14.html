Beta build 14 adds an in-app App Store workflow, reusable captured screens, and AI-generated store assets for the current project.

New App Store Workspace

  • A new App Store tab lets you review your generated description, screenshots, and icon in one place.
  • App Store assets are saved with the project and can be opened in Finder or exported as a ZIP.

Captured Screen Library

  • You can now capture screens from Development, keep them in a saved library, rename them, and manage them from a dedicated screen manager.
  • Saved captures can be reused when generating App Store screenshots, which makes it easier to turn real app screens into store-ready assets.

AI-Generated Store Assets

  • 10x can now generate an App Store description, compose screenshot sets from captured screens, and create a polished App Store icon concept for the project.
  • The App Store asset workflow is built around your actual project content instead of generic placeholders.

Notes

  • For the best screenshot results, capture a few strong screens in Development before generating App Store assets.
  • Existing projects can start using the App Store workflow without migration.
