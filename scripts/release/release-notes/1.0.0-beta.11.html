Beta build 11 is a small polish release that makes a few controls easier to click and removes frustrating dead zones in the interface.

Easier Interactions

  • The main actions on the sign-in flow now respond across the full button shape instead of just the text, so it is easier to continue through onboarding.
  • Settings items are easier to select because the whole row is now clickable.
  • This build focuses on interaction polish rather than new features.

Notes

  • No migration or cleanup is required.
  • Your existing projects and account data are unaffected.
