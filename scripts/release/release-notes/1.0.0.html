This release makes sign-in and billing feel more direct, with faster access to plans and packs when you run out of credits.

Simpler Sign-In

  • Apple sign-in now follows the same browser-based flow across the app, which makes the authentication experience more consistent.
  • Sign-in setup is now aligned more closely with the hosted auth configuration used in production.

Easier Plans And Packs Access

  • If you run out of credits, 10x now points you straight to Plans and Packs from chat and the home screen instead of leaving you at a dead end.
  • The billing catalog is easier to reopen from around the app, so upgrading or topping up takes fewer steps.

Clearer Credit And Promo Flow

  • The signup credit flow is now simpler and more direct, with clearer states when free credits have already been used on the current Mac.
  • Billing messaging throughout the app has been cleaned up so availability reflects the current build, not older beta-specific wording.

More Reliable Billing Updates

  • After checkout or billing portal changes, balances, invoices, and plan status refresh more reliably when you return to the app.
  • This should reduce cases where billing looked stale right after a purchase or subscription change.

Notes

  • The downloads experience is also more forgiving now: if a stable build has not been published yet, the site can fall back to the latest beta instead of showing a broken state.
  • Existing accounts and projects continue to work without migration.
